## 博客特色

这个博客使用 [Hexo](https://hexo.io/) 静态博客框架搭建，具有以下特点：

- 🚀 快速加载
- 📱 响应式设计
- 🔍 SEO 友好
- 🎨 简洁美观

## 开始使用

如果你也想搭建类似的博客，可以查看 [Hexo 官方文档](https://hexo.io/docs/)。

### 常用命令

```bash
# 创建新文章
hexo new "文章标题"

# 启动本地服务器
hexo server

# 生成静态文件
hexo generate

# 部署到 GitHub Pages
hexo deploy
```
